ENRUTAMIENTO EN ANGULAR


Definición:
El enrutamiento en Angular permite la navegación entre diferentes vistas o componentes sin recargar la página. Se maneja a través del módulo RouterModule y se configura en un array de rutas.

Definición de rutas:
En la carpeta src/app debe existir el fichero app.routes.ts en el cual se definirán las distintas rutas que se van a utilizar en la aplicación:

  • - Se establece una constante de tipo Routes, que es parte del modulo "@angular/router" y se utiliza para definir los enrutamientos.
  • - Se establece un array de rutas (URL) definidas con la propiedad path.
  • - La ruta vacía ("") indica la ruta por defecto al arrancar. El término redirectTo indica la ruta a la que se redirige. El término pathMatch: "full" indica que la ruta del navegador debe coincidir estrictamente con la ruta vacía definida (""). Si se omite, el enrutamiento puede dejar de funcionar.
  • - Se establece el componente al que apunta cada ruta con el término component.
  • - La ruta "**" es la ruta comodin (wildcard), que captura cualquier ruta no definida y la redirige a la ruta que se indique.
            
            export const rutas: Routes = [
                {path: "", redirectTo: "rutaPorDefecto", pathMatch: "full"},
                {path: "rutaPorDefecto", component: ComponentePorDefecto},
                {path: "rutaComponenteX", component: ComponenteX},
                {path: "rutaErrores", component: ComponenteErrores},
                {path: "**", redirectTo: "rutaErrores"}
            ];

Configuración:
En el fichero app.config.ts debe configurarse el enrutador provideRouter como un servicio/proveedor global de la aplicación: 

                
                export const appConfig: ApplicationConfig = {
                providers: [
                  (...otros servicios...), 
                  provideRouter(rutas)
                ]};

EJEMPLO 1: Enrutamiento de menú

En el desarrollo de componentes anteriores y/o sprint, manejábamos las opciones del menú de navegación a través de la directiva ngIf. Aplicando el enrutamiento de Angular, se simplifica todo el código anterior, de manera que cada opción del menú se referencia (href) a una ruta que apunta a cada componente (previamente definida en app.routes.ts).
