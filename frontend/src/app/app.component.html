COMPONENTES

Definición:
Un componente es un bloque o módulo que encapsula una determinada funcionalidad de la interfaz de usuario. Esto permite que se pueda reutilizar en distintas parte de la aplicación sin tener que volver a codificar.

Cada componente tiene tres elementos clave:
  • Clase (TypeScript): Elemento que contiene la lógica del componente
  • Plantilla (HTML): Elemento que define la estructura y el diseño visual del componente
  • Estilo (CSS): Elemento que aporta el estilo visual del componente
