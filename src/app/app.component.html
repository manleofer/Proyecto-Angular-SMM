DIRECTIVAS

Definición:
Las directivas son elementos que se aplican a etiquetas HTML y añaden funcionalidad a la etiqueta donde se aplican.
Sirven para modificar la estructura del DOM y para modificar la apariencia de las aplicaciones.
Tipos:
Estructurales, de Atributos y Componentes.

Directivas estructurales

Descripción

Directiva ngIf

Definición:
Es una directiva estructural que permite incluir o excluir un elemento del DOM en función de una expresión booleana. Si la expresión evaluada es true, el contenido del elemento es insertado en el DOM; si es false, el contenido es eliminado del DOM.
Ejemplo:
Al hacer click en Directivas te muestra todas las directivas, esto esta hecho como ejemplo de ngIf

Directiva ngSwitch

Definición:
Es una directiva estructural similar al ngIf y se utiliza como el switch en lógica de programación. En esta directiva se pueden crear los diferentes casos que deseamos evaluar y cuando se cumple la condición esperada, oculta o muestra el HTML.
Ejemplo:

Directivas de atributos

Descripción

Directiva ngStyle

Definición:
Es una directiva de atributo que permite vincular un objeto a los estilos CSS de un elemento HTML de manera dinámica.
Ejemplo:

Recibe un conjunto de propiedades CSS como un valor, y Agular actualiza los estilos del elemento en el DOM en función de las propiedades.(Aquí se vería aplicado y por eso sale el fondo azul con letras en rojo).

Directiva ngClass

Definición:
Es una directiva de atributo que permite aplicar, añadir o eliminar clases CSS a elementos HTML.
Ejemplo:
Al pulsar un botón, la clase CSS del párrafo cambia.
