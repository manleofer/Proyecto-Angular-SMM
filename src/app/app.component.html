DIRECTIVAS

Definición:
Las directivas son elementos que se aplican a etiquetas HTML y añaden funcionalidad a la etiqueta donde se aplican.
Sirven para modificar la estructura del DOM y para modificar la apariencia de las aplicaciones.
Tipos:
Estructurales, de Atributos y de Componentes.

Directivas estructurales

Descripción

Directiva ngFor

Definición:
Es una directiva estructural que permite iterar sobre una colección de datos (arrays o listas) y poder manejar una instancia o elemento. Aplica un mismo bloque de código para cada interación. Osea, hace que un elemento HMTL pueda generarse como un bucle
Ejemplo:
Al añadir tareas en el siguiente formulario, éstas se almacenan en un array y se muestran en una tabla. La directiva ngFor permite recorrer el array para generar cada fila de la tabla y asignarle un índice según el orden en el que se han ido añadiendo.

Directiva ngSwitch

Definición:
Es una directiva estructural similar al ngIf y se utiliza como el switch en lógica de programación. En esta directiva se pueden crear los diferentes casos que deseamos evaluar y cuando se cumple la condición esperada, oculta o muestra el HTML.

Directivas de atributos

Descripción

Directiva ngClass

Definición:
Es una directiva de atributo que permite aplicar, añadir o eliminar clases CSS a elementos HTML.
Ejemplo:
Al pulsar un botón, la clase CSS del párrafo cambia.

Directiva ngmodel

Definición:
Es una directiva de atributo que generaa un enlace bidireccional entre el modelo (datos especificados en el componente) y la vista (el HTML). Permite que cualquier cambio en el valor de un campo de entrada en el HTML actualice automáticamente el valor en el componente, y viceversa
Ejemplo:
Al agregar registros en el siguiente formulario, cada input se corresponde con una propiedad establecida en el componente, de forma que se enlaza con un atributo específico del objeto "registro" que genera la aplicación. Se combina la directiva ngIf para validar los datos del formulario y mostrar un mensaje si algo es incorrecto. Al enviar los datos, estos se muestra en un mensaje de confirmación.
